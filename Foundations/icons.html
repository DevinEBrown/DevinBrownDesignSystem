Icons

The icons are part of the Font Awesome system. They should never be used at less that 18px height and width. Only use icons in Cinnamon or Coffee.

Social Media

Social media icons serve as hyperlinks to brands corresponding social media profiles. Socail media icons should never be used at less than 20px. Only use social media icons in Latte or Coffee.

Glyphs

Glyphs should be used with actionable components that also contan content. They should never be used at less than 20px height and width. Only use glyphs in Cinnamon or Coffee.
